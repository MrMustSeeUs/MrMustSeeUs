### Featured Projects

**[NeciAI](https://github.com/MrMustSeeUs/neciai)** — Intelligent Financial Data Analysis Platform
Full-stack financial platform with anomaly detection, PDF/Excel report generation, and real-time dashboards. Independently deployed across three cloud platforms (Railway, Vercel, Neon) with environment-variable-driven secrets management, hashed credentials, and account lockout protection — built to reflect production security practices, not just a working demo.
`C# · ASP.NET Core 8 · React 18 · PostgreSQL · Docker`
[Live App](https://neciai-ai.vercel.app) · [API Docs](https://neciai-production.up.railway.app/swagger) · [Source](https://github.com/MrMustSeeUs/neciai)

**[AI Resume Screener](https://github.com/MrMustSeeUs/ai-resume-screener)** — LLM-Powered Resume Matching
Scores resumes against job descriptions using the Anthropic Claude API, with Pydantic-validated inputs and an automated test suite running on every commit via GitHub Actions.
`Python · Streamlit · Claude API · pytest · Docker · GitHub Actions`
[Source](https://github.com/MrMustSeeUs/ai-resume-screener)

**[AI Support Chatbot](https://github.com/MrMustSeeUs/ai-support-chatbot)** — Knowledge-Base-Grounded Support Bot
A 24/7 support assistant grounded in a custom knowledge base, deployed at the edge via Cloudflare Workers, with explicit prompt-injection filtering and secrets management baked into the design.
`Python · Claude API · Gradio · Cloudflare Workers`
[Source](https://github.com/MrMustSeeUs/ai-support-chatbot)

**[Taniti Island](https://github.com/MrMustSeeUs/TanitiPrototype)** — Responsive Tourism Site
